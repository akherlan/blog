Tema Hugo Lithium Tidak Responsif

Terakhir diperbarui 26 Mar 2019

Ini merupakan tulisan dokumentasi saya untuk permasalahan tema Lithium yang tidak responsif (saat tulisan ini mulai ditulis). Ukuran gambar tidak menyesuaikan ukuran layar. Sangat fatal ketika dibuka dari ponsel. Padahal sebagian besar akses internet hari ini adalah melalui ponsel. Alamak!

Awalnya saya tanya di grup GNU/R Indonesia, walaupun ini tidak ada kaitannya sama sekali dengan pemrograman R. Tapi karena saya ngeblog dari R menggunakan paket blogdown, jadi saya coba tanyakan saja. Cocoklogi. Hihihi… Ada saran, ganti tema yang responsif saja, atau tambahkan skrip bootstrap di dalamnya, alamak! Nggak ngerti!

Saya tidak menyerah, tetap searching… Seperti biasa. Ketemu!

Malalui repositori github pembuat awalnya ternyata sudah dibuat responsif. Saya mengira barangkali ini sudah mengalami pemutakhiran kode CSS dari sebelumnya.

Namun ada satu kendala lagi. Saya harus mencari bagian mana yang diubah. Saya tidak langsung mengganti tema Lithium yang saya gunakan (dari yihui) dengan tema Lithium yang dimodifikasi oleh pembuat awalnya (jrutheiser).

Untungnya ketemu, ada di dalam main.css, seperti berikut:

img {
  max-width: 100%;
  height: auto;
}

Voila! Ukuran gambar di blog ini sudah menyesuaikan ukuran layar. Sudah. Ternyata begitu saja.

Sekian. Terima kasih.
